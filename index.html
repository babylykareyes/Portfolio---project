Vector Art

gallery

Vector artwork is a term that describes any made with vector illustration software like Adobe Illustrator. Vector artwork is built from vector graphics, which are images created with mathematical formulas. In comparison, raster art (also referred to as bitmaps or raster images) is created with colorized pixels.
